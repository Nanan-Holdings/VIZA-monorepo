# India — Digital Personal Data Protection Act, 2023 (DPDPA)

> **Status:** needs work. Some DPDPA obligations effective on a
> rolling basis from 2025; full counsel review required.

## Applicability

- VIZA processes Indian residents' data when they apply for visas via
  the India e-Visa runner (`indianvisaonline.gov.in`) and when Indian
  residents apply to other destinations through VIZA.
- DPDPA applies to processing of digital personal data in India and to
  processing outside India in connection with offering services in
  India.

## Key obligations

| Obligation | VIZA posture |
|---|---|
| Notice + consent (S.5) | Privacy Policy + LEGAL-002 signup gate. |
| Consent manager interaction (S.6) | Open: India is rolling out a consent-manager registry; integration pending counsel guidance. |
| Significant Data Fiduciary | Triggered above thresholds notified by the government; not yet at scale. Monitor. |
| Cross-border restriction (S.16) | Permits transfers to all countries except those notified as restricted; default Singapore + global subprocessors are permissible at present. |
| Children's data (S.9) | Verifiable parental consent for minors under 18. Guardian-mode applications need to be flagged accordingly. |
| Breach notification | To Data Protection Board on notification timelines (rules pending). |
