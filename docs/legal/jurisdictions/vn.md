# Vietnam — Decree 13/2023 on Personal Data Protection

> **Status:** needs work. Cross-border filing required for routine
> processing of Vietnamese residents' data.

## Applicability

VIZA handles Vietnamese residents' data through the VN e-Visa runner
and inbound applications. Decree 13/2023 (effective July 2023) applies
to organisations outside Vietnam that process Vietnamese residents'
data.

## Key obligations

| Obligation | VIZA posture |
|---|---|
| Consent + notice | Captured at signup. |
| Cross-border data transfer impact assessment (TIA) and dossier filing | Required before routine cross-border transfer. **Open:** TIA template + Ministry of Public Security filing. |
| Sensitive data (S.2) | Stricter rules; passport + biometric photo treated as such. |
| Data subject rights | Mirrored by LEGAL-004 export + delete. |
| Breach response | 72-hour MPS notification. |

## Gaps

- TIA template + signed dossier filed with MPS.
- Consent text adapted to Decree 13's specific list of permitted purposes.
