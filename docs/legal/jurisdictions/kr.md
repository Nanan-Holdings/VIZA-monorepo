# South Korea — PIPA

> **Status:** needs work. PIPA cross-border consent and cookie/SDK

## Applicability

VIZA processes Korean residents' data through the KR C-3-9 runner
and inbound applications. PIPA (revised 2023) applies broadly to
processing of Korean personal information.

## Key obligations

| Obligation | VIZA posture |
|---|---|
| Notice + consent (Art. 15) | Captured. PIPA requires opt-in consent and separate opt-in for cross-border transfers (Art. 28-8). |
| Domestic representative | Required for offshore controllers handling 1m+ users / sensitive data; VIZA below thresholds today, monitor. |
| Cross-border transfer | Requires explicit consent + notice of recipient country and purpose. Open: not yet wired into the consent gate. |
| Breach notification | 72 hours to PIPC + affected users. |
