## 📖 三页流程说明

### 第 1 页：首页 (Home)
```
特征：有红色 "Apply now" 按钮
操作：
  1. 等待插件初始化 (2-3 秒)
  2. 看到浮动面板
  3. 点击红色 Apply 按钮
```

### 第 2 页：条款页面 (Disclaimer)
```
特征：需要阅读条款和勾选同意
操作：
  1. 看到提示："请阅读并勾选同意条款"
  2. 向下滚动阅读全部内容
  3. 插件自动在 90% 时勾选
  4. Next 按钮自动启用
  5. 点击 Next 继续
```

### 第 3 页：表单页面 (Form)
```
特征：多个输入字段（姓名、邮箱、护照等）
操作：
  1. 看到中文标签在每个字段旁边
  2. 点击浮动面板的"🟢 一键填表"
  3. 所有字段自动填充✓
  4. 检查数据正确性
  5. 点击 Submit 提交
```

## 🆘 常见问题速查

### ❓ 浮动面板看不到？
```
检查:
  1. URL 是否为 https://evisa.gov.vn
  2. 刷新页面 (F5)
  3. 打开控制台 (F12) 查看错误
  4. 禁用/重新启用扩展
```

### ❓ Apply 按钮没有高亮？
```
检查:
  1. 是否在首页 (应该看到大的红色按钮)
  2. 等待 2-3 秒让插件初始化
  3. 打开控制台看是否有错误信息
  4. 尝试刷新页面
```

### ❓ 自动填表没有工作？
```
检查:
  1. 是否在表单页面 (有多个输入字段)
  2. 点击"一键填表"后等待 1 秒
  3. 打开控制台看反馈信息
  4. 检查是否显示"✅ 已填充 X 个字段"
```

### ❓ 看到错误信息？
```
常见错误及解决:
  - 400 Bad Request: 重新加载页面
  - 401 Unauthorized: 认证过期，使用官方途径登录
  - undefined function: 刷新页面
  - 字段未填充: 等待 1-2 秒后重试
```
