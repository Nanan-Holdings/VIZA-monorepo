## 🚀 如何使用新版本

### 安装步骤
1. 打开 Chrome 或 Edge 浏览器
2. 在地址栏输入 `chrome://extensions/` 然后按 Enter
3. 打开右上角的 **"开发者模式"** 开关
4. 点击 **"加载已解压的扩展程序"**
5. 选择 `vietnam-visa-helper-v1` 文件夹
6. 完成！插件已安装

### 三步快速使用

#### 第1步：打开越南e-visa网站
🌐 访问 https://evisa.gov.vn  
你会看到右上角出现浮动面板 "🇻🇳 v1.2.1"

#### 第2步：点击红色Apply按钮
👆 首页的Apply按钮会被红色高亮  
点击它进入申请流程

- **Disclaimer页面** - 滚动阅读，插件会自动处理
- **表单页面** - 点击浮动面板的"一键填表"按钮
- **自动填充** - 所有字段会以你的信息自动填充

## 💡 使用贴士

### 💼 最佳实践

✅ **推荐做法**：
1. 在首页先确认Apply按钮已被高亮
2. 阅读Disclaimer并让插件自动处理
3. 在表单页点击"一键填表"
4. 检查自动填充是否正确
5. 如有不对的地方手动修正后提交

❌ **避免**：
1. 不要在页面加载前点击按钮，等待插件初始化
2. 不要手动修改已自动填充的字段然后再点"一键填表"（会覆盖）
3. 不要在其他网站使用这个插件（仅限evisa.gov.vn）

### 🔧 遇到问题时的排查

**问题：浮动面板不出现**  
➜ 解决方案：
1. 确保网站是 https://evisa.gov.vn
2. 打开浏览器开发者工具 (F12)，查看Console标签
3. 应该看到"🇻🇳 越南签证助手 v1.2.1 已激活"的消息
4. 如果没有，尝试刷新页面

**问题：Apply按钮没有被高亮**  
➜ 解决方案：
1. 确保你在网站首页
2. 等待2-3秒，让插件完成检测
3. 页面上应该有一个红色的"Apply now"按钮
4. 如果找不到，可能是网站结构改变了

**问题：自动填表没有工作**  
➜ 解决方案：
1. 确保你在表单页面
2. 打开Console (F12 -> Console标签)
3. 点击"一键填表"按钮
4. 应该看到"✅ 已填充 X 个字段"的消息
5. 如果有错误信息，请记录下来用于调试

**问题：400/401 错误**  
➜ 解决方案：
1. 这些是网站服务器的错误，不是插件的问题
2. 通常意味着：
   - 认证信息过期
   - 网络连接问题
   - 网站暂时不可用
3. 尝试刷新页面或稍后重试
