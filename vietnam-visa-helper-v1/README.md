# 越南签证助手 v1.1 - 完整引导版本

一个专为中文用户设计的浏览器扩展插件，帮助简化越南电子签证(e-Visa)申请流程。

## 🎯 核心功能

### 1. **智能页面检测与引导**
- 自动识别当前所在页面（首页、申请页、条款页、表单页）
- 根据页面类型提供相应的指导和帮助
- 实时页面变化监听

### 2. **Apply按钮引导** 
- 高亮显示申请按钮，带有脉冲动画
- 提供顶部指导横幅
- 自动滚动定位到申请按钮
- 让用户清楚地知道要点击什么

### 3. **Disclaimer页面智能处理**
- 检测是否用户已阅读完整条款（滚动到底部）
- 滚动进度提示和动画
- 自动勾选同意复选框
- 智能启用/禁用下一步按钮
- 当用户完成阅读时自动高亮Next按钮

### 4. **一键自动填表**
- 点击"一键自动填表"按钮自动填充所有表单
- 支持文本输入框、下拉列表、单选框、复选框
- 实时事件触发（input、change事件）
- 自动标记已填充字段（带绿色✓标记）

### 5. **完整的中文翻译与说明**
- 每个字段都有中文标签
- 详细的填写提示和说明
- 晦涩选项的详细解释
- 格式正确的示例
- 点击字段旁的❓问号图标查看完整说明

### 6. **浮动控制面板**
- 右上角始终可见的控制面板
- 支持最小化/展开
- 按钮说明：
  - 🔴/🟢 **一键自动填表** - 启用/禁用自动填充
  - 📋 **查看数据** - 查看已存储的所有信息
  - 💡 **切换提示** - 显示/隐藏所有字段提示
  - 🆘 **需要帮助** - 显示针对当前页面的帮助指南

### 7. **傻瓜式用户体验**
- 大量使用emoji和颜色提示
- 清晰的通知和反馈
- 每一步都有引导
- 成功/错误的视觉反馈

## 📋 完整申请流程引导

### 第1步：点击申请
1. 访问 https://evisa.gov.vn
2. 插件自动激活，显示绿色高亮的"Apply"按钮
3. 跟随指导，点击按钮

### 第2步：阅读条款
1. 进入条款页面（Disclaimer）
2. 向下滚动阅读完整条款
3. 插件检测滚动进度
4. 自动勾选"我同意"复选框
5. 点击"Next"继续

### 第3步：填写表单
1. 进入表单填写页面
2. 每个字段都有中文标签和❓说明
3. 点击"一键自动填表"按钮自动填充信息
4. 可以手动修改任何字段
5. 查看数据确认信息正确

### 第4步：提交申请
1. 检查所有字段是否正确
2. 点击表单的提交按钮
3. 保存申请号以便追踪

## 🔧 技术实现

- **Manifest V3** - 最新的Chrome扩展标准
- **Content Script** - 在页面上注入引导和自动填表功能
- **Background Service Worker** - 管理用户数据和字段映射
- **本地存储** - 所有数据完全存储在本地，无云同步

## 💾 数据隐私

❌ **不做的事情：**
- 不上传任何数据到服务器
- 不追踪用户活动
- 不保存敏感信息到云端
- 不与第三方共享数据

✅ **所有数据：**
- 完全存储在本地浏览器
- 仅用于自动填表
- 用户可随时清除

## 🚀 安装步骤

### 方法1：本地安装（开发者模式）
1. 克隆或下载此项目
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 启用右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择包含 `manifest.json` 的文件夹
6. 完成！该扩展现在会在 evisa.gov.vn 上激活

## 🎓 使用提示

1. **始终检查字段：** 虽然有自动填表，但建议逐一检查字段确保正确
2. **保存申请号：** 提交后，务必保存申请号以便追踪申请状态
3. **提前准备护照信息：** 在申请前准备好护照、出生证明等文件扫描件
4. **检查有效期：** 确保护照在越南停留期间+6个月内有效
5. **邮箱验证：** 确保邮箱能接收越南官方邮件（检查垃圾箱）

## ⚙️ 设置和自定义

### 修改默认信息
编辑 `background.js` 中的 `userData` 对象：

```javascript
const userData = {
  personalInfo: {
    full_name: "你的名字",
    surname: "你的姓氏",
    // ... 其他信息
  },
  // ...
};
```

### 添加新字段
在 `background.js` 的 `fieldMappings` 中添加：

```javascript
new_field: {
  label_cn: "中文标签",
  hint: "详细说明",
  example: "示例",
  options: null // 如果是下拉菜单，设置为{value: "中文"}
}
```

## 🐛 常见问题

**Q: 插件不工作怎么办？**
A: 
1. 确认你在 evisa.gov.vn 网站上
2. 检查Content Script是否注入（浏览器控制台查看日志）
3. 刷新页面重新加载插件
4. 清除浏览器缓存后重试

**Q: 自动填表为什么没有填充所有字段？**
A: 
1. 该网站的字段可能有特殊名称，插件未能识别
2. 某些字段可能有动态加载
3. 尝试手动在浏览器控制台查看字段属性

**Q: 如何更新我的信息？**
A: 编辑 `background.js` 中的 `userData` 对象，然后在 `chrome://extensions` 重新加载扩展

**Q: 支持多个申请人吗？**
A: 目前不支持，但可以通过编辑代码添加多个用户配置
