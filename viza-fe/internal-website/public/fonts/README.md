# Sofia Pro Font Files

Please add the following Sofia Pro font files to this directory:

- SofiaPro-Light.woff2
- SofiaPro-Regular.woff2
- SofiaPro-Medium.woff2
- SofiaPro-SemiBold.woff2
- SofiaPro-Bold.woff2

Sofia Pro is a commercial font. You can purchase it from:
- https://www.myfonts.com/collections/sofia-pro-font-mostardesign-studio

The font configuration is set up in `app/layout.tsx` to use these font files.
