Keyboard & focus rules
Focus ring

1px brand-500 ring via focus-visible:ring-1. Never removed — only repositioned.

Tab order

DOM order matches visual order. Never use positive tabindex.

Esc closes overlays

Dialog, sheet, popover, dropdown, tooltip — all dismiss on Esc. Returns focus to trigger.

Arrow keys in menus

Dropdown, select, tabs, radio — ↑↓ moves, Enter activates, Home/End jump.

Touch target · 44×44

Minimum hit area on mobile. Flow CTA is 48×— meets it. Ghost button 40× uses padding extension.

Body ≥ 16px on mobile

Prevents iOS auto-zoom on input focus. Labels can be 12px; body text cannot.

Screen-reader-only labels

Icon-only buttons wrap their icon with <span class="sr-only"> text.

Reduced motion

Motion's useReducedMotion() swaps y/scale animations for opacity-only fades.
